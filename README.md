## Näin Github toimii

### Lisää tekemäsi muutokset
Lisää tiedostot: `git add .`

Kerro mitä ole tehnyt: `git commit -m "Kirjoita tänne jotain"`

Päivitä remote: `git push origin master`

### Hae viimeiset muutokset
`git pull origin master`
